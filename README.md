# Acadify – AI Powered Study Assistant

Acadify is an AI-powered learning assistant designed to help students create structured study plans and discover relevant learning resources quickly.

The platform analyzes user study goals and generates personalized roadmaps along with recommended YouTube videos for efficient learning.

---

## Features

AI Study Roadmap Generation  
YouTube Learning Video Recommendations  
Chat-Based Study Planning  
Chat History Management  
PDF Notes Upload Support  
Authentication System (Email, Google, GitHub)  
Interactive Dashboard UI  
Modern Animated Landing Page

## Installation

Clone the repository

```
git clone https://github.com/yourusername/acadify-ai-learning-assistant.git
```

Install backend dependencies

```
cd server
npm install
```

Install frontend dependencies

```
cd client
npm install
```
---

## Running the Project

Start backend

```
cd server
npm run dev
```

Start frontend

```
cd client
npm run dev
```
